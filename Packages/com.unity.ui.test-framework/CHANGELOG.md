# Changelog
## [6.5.0] - 2026-01-21
- Change package version to 6.5.0 to match associated Unity Editor version.
- Add support for simulating interactions with World Space UI.
- Add property to fetch the list of menu items when using ContextMenuSimulator or PopupMenuSimulator.
- Update documentation with recommendations for using the panelSize property.

## [1.0.0] - 2025-07-09
- Move com.unity.ui.test-framework to Packages.
- Update documentation with recommendations for using TypingText API on platforms that rely on a TouchScreenKeyboard.
