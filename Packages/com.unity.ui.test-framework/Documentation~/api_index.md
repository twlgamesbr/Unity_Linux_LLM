# Scripting API reference for UI Test Framework

This section of the documentation provides detailed information about the UI Test Framework scripting API. To effectively use this information, you should be familiar with the basic concepts and practices of scripting in Unity, as explained in the Scripting section of our manual.

The scripting reference is organized by classes, which are described along with their methods, properties, and other relevant information.

APIs are grouped by their respective namespaces and can be selected from the sidebar on the left.
