# Customize test fixtures with UITestComponent

[UITestComponent](xref:UnityEngine.UIElements.TestFramework.UITestComponent) is a powerful feature of the UI Test Framework that allows you to customize and extend the behavior of your test fixtures:

| Topic | Description |
|:---|:---|
| [Add styles during testing](xref:add-styles-during-testing) | Add styles to elements during testing. |
| [Clean up objects after tests](xref:clean-up-objects-after-tests) | Dispose or destroy objects at the end of each test. |
| [Simulate context menu actions](xref:simulate-context-menu-actions) | Simulate user interactions with context menus. |
| [Simulate popup menu actions](xref:simulate-popup-menu-actions) | Simulate user interactions with popup menus. |
| [Create reusable test components](xref:create-reusable-test-components) | Add your own test component. |
