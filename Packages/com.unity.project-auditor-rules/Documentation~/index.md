# Project Auditor Rules package

Project Auditor is a static analysis tool for Unity projects that is built in to the Unity Editor. This package contains the rules that Project Auditor uses as the basis for recommending improvements to a project. You don't have to interact with this package other than installing it as a dependency of Project Auditor.

## Installation

To launch Project Auditor in the Editor, go to **Window** > **Analysis** > **Project Auditor**. On launch, Project Auditor prompts you to install this package before it can analyze your project.

Click the following link to open the Editor and install the Project Auditor Rules package directly: [Open the Unity Editor and install the Project Auditor Rules](com.unity3d.kharma:upmpackage/com.unity.project-auditor-rules).

Alternatively, follow the instructions in the [Add and remove UPM packages or feature sets](xref:um-upm-ui-actions) documentation. You can install the package using the [package registry list](xref:um-upm-ui-install), or by using `com.unity.project-auditor-rules` as the name when [adding a registry package by name](xref:um-upm-ui-quick).
