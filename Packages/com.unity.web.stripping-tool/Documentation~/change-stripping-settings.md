# Change the active stripping settings

Check what settings are active on a build, change the active settings, or disable the settings.

## Check the active stripping settings

To check the active stripping settings on a build:

1. Go to **Window** > **Web Optimization** > **Submodule Stripping**.
1. Refer to the **Stripping Settings** property.

Alternatively:

1. In the **Project window**, open the **Assets** folder.
1. Right-click and select **Web Optimization** > **Reveal Active Submodule Stripping Settings**.

You can also check the active settings from the **Assets** menu.

## Change the active stripping settings

To change the active stripping settings on a build:

1. In the **Project window**, open the **Assets** folder.
1. Right-click on the stripping settings asset and select **Web Optimization** > **Set as Active Submodule Stripping Settings**.

You can also change the active settings from the **Submodule Stripping** window or the **Assets** menu.

## Disable stripping settings

To disable all stripping settings:

1. Go to **Window** > **Submodule Stripping**.
1. In the **Stripping Settings** property, use the object picker to select **None**.
