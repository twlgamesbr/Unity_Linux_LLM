# Installation

To install the Web Stripping Tool package in your project, follow the instructions to [Install a UPM package from a registry](xref:upm-ui-install).

## Requirements

This version of the Web Stripping Tool package is supported by the following versions of the Unity Editor:

- 6000.1.0a4 and later

The Web Build Support module is also required. To add it through the Unity Hub, refer to [Add modules to the Unity Editor](https://docs.unity3d.com/hub/manual/AddModules.html).
