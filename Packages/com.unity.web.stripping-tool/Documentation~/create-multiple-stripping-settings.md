# Create multiple stripping settings

You might want multiple stripping settings to use in different scenarios. To do so, you need to create new stripping assets.

To create a stripping asset:

* Go to **Assets** > **Create** > **Web Optimization** > **Submodule Stripping Settings**.

You can also create stripping assets from the [Project Window](xref:ProjectView).
