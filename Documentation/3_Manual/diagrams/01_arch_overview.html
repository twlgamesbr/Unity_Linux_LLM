NPCSystem Architecture — WebGL Multiplayer Dialogue

Unity • LocalAI • Qdrant • Supabase • Datadog

Core Pipeline

  • • Player Input → Dialogue Manager → LLM
  • • LLM → RAG (Qdrant vector DB)
  • • Response → Game State Update
  • • Server-authoritative state sync

Backend Stack

  • • LocalAI — LLM inference + embeddings (:8080)
  • • Qdrant — Vector database for RAG (:6333)
  • • Supabase — Auth + PostgREST persistence (:8091-8092)
  • • Datadog — Metrics and APM traces (:8125-8126)

WebGL Constraints

  • • IL2CPP AOT compilation (no JIT runtime)
  • • 2 GB WASM heap memory limit
  • • Server-authoritative networking model
  • • WebSocket transport only (no raw TCP)
